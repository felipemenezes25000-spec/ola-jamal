# Mercado Pago — Resumo das cláusulas de processamento de dados (LGPD)

**Fonte:** [Termos e Condições para Desenvolvedores do Mercado Pago](https://www.mercadopago.com.br/developers/pt/docs/resources/legal/terms-and-conditions) (Cláusula 6)

O Mercado Pago **não possui DPA separado**. As obrigações de processamento de dados estão incorporadas nos Termos e Condições para Desenvolvedores. O aceite ocorre ao utilizar a API.

---

## Cláusula 6 — Principais pontos

| Aspecto | Disposição |
|---------|-------------|
| **Papéis** | Usuário do Mercado Pago = Controlador; Desenvolvedor = Operador (processador). Ambos se comprometem com a LGPD. |
| **Finalidade** | Processar apenas conforme instruções do Controlador e para cumprir os Termos. |
| **Subprocessadores** | Garantir que subprocessadores limitem uso às finalidades, mantenham dados seguros e cumpram LGPD. |
| **Autoridades** | Notificar Mercado Pago imediatamente de comunicações de autoridades competentes. |
| **Incidentes** | Notificar Mercado Pago (privacidad@mercadolibre.com) sobre Incidente de Privacidade. |
| **Segurança** | Medidas técnicas e organizacionais adequadas (Cláusula 10). |
| **Retenção** | Não armazenar além do necessário; exceções: lei ou autorização do titular. |

---

## Incidente de segurança (Cláusula 10)

- Notificar em **até 24 horas** via [formulário](https://docs.google.com/forms/d/e/1FAIpQLSdnyVbe99JjH-_wafeFS7yI097YK0a2_1UcY4-agGR6gDAL0Q/viewform)
- Declarações públicas: somente com anuência prévia do Mercado Pago

---

## Conclusão

O uso da API do Mercado Pago implica aceite dos Termos, que já incluem cláusulas compatíveis com LGPD. **Não é necessário assinar DPA separado.**
